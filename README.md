## 购物网站实现了以下功能

1.用户注册

2.用户登录，退出登录

3.页面展示

4.商品列表

5.搜索商品

6.商品详情

7.购物车

8.提交订单

9.支付

1.订单查询

## 安装依赖包
```
npm install
```

### 在终端输入以下代码运行文件
```
npm run serve
```

### 打包
```
npm run build
```
